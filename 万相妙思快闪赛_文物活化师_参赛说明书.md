## 二、作品简介

### 核心定位

AI文物活化师是一个基于**阿里通义万相 Wan 2.7** API 的智能体Skill，专门为文博机构、文旅从业者、文化遗产爱好者设计。通过AI影像生成能力，将静态文物照片转化为**动态影像、场景还原、历史穿越视频**，让沉睡千年的文物"活"起来。

### 一句话描述

> **输入一张文物照片 → AI自动生成文物活化动态视频 → 可直接用于展览、宣传、教育**

### 核心价值

- **零门槛活化**：无需专业视频制作技能，一张照片即可生成文物动态视频
- **多场景适配**：博物馆展厅、社媒传播、教育科普、文旅推广全覆盖
- **可批量复用**：同一文物可生成多种风格（纪录片风、穿越风、奇幻风等）
- **文博专业级**：针对文物特点优化的Prompt策略，确保历史准确性与视觉美感兼顾

### 解决的问题

| 痛点 | 现状 | AI文物活化师的方案 |
|------|------|-------------------|
| ❌ 文物展示方式单一 | 静态展柜+文字说明 | 动态影像让文物"动"起来 |
| ❌ 视频制作成本高 | 一条文物宣传视频数万元 | 一张照片几分钟生成 |
| ❌ 年轻人不感兴趣 | 传统文化传播难触达Z世代 | AI生成的奇幻/穿越风格吸引年轻人 |
| ❌ 中小博物馆无力制作 | 缺预算、缺技术、缺人力 | 低成本、零门槛、一键生成 |

### 5.3 智能Prompt引擎逻辑

```python
def generate_relic_prompt(relic_info: dict, style: str) -> str:
    """
    根据文物信息和风格生成优化Prompt
    
    Args:
        relic_info: {
            "name": "汝窑天青釉洗",
            "dynasty": "北宋",
            "type": "瓷器",
            "material": "天青釉瓷",
            "features": "天青色釉面，细密开片纹",
            "background": "北宋宫廷审美巅峰"
        }
        style: "纪录片风" | "奇幻穿越风" | "古代复原风" | "粒子光效风" | "博物馆展示风"
    """
    
    # 基础描述层
    base = f"{relic_info['name']}, {relic_info['dynasty']} dynasty {relic_info['type']}, {relic_info['material']}, {relic_info['features']}"
    
    # 风格层
    style_prompts = {
        "纪录片风": "cinematic documentary style, slow motion, museum spotlight, shallow depth of field, warm ambient light",
        "奇幻穿越风": "magical awakening, golden particles swirling, ethereal glow, time travel transformation, fantasy atmosphere",
        "古代复原风": "historical recreation, period-accurate setting, craftsmen working, ancient workshop, warm candlelight",
        "粒子光效风": "particle effects, light trails, golden dust floating, ethereal glow, luminous transformation",
        "博物馆展示风": "clean museum display, soft rotating spotlight, dark background, professional showcase, 4K quality"
    }
    
    # 质量层
    quality = "professional cultural heritage photography, 4K resolution, smooth camera movement, historically accurate details"
    
    return f"{base}, {style_prompts[style]}, {quality}"
```

### 5.4 历史准确性校验层（⭐ 新增）

针对博物馆级专业使用场景，在Prompt引擎之上叠加一层**朝代/纹饰白名单校验**，防止Wan 2.7 生成历史错误内容：

```python
# 朝代-纹饰白名单（可扩展）
DYNASTY_PATTERN_WHITELIST = {
    "商": ["饕餮纹", "云雷纹", "蝉纹", "夔纹", "涡纹"],
    "周": ["窃曲纹", "重环纹", "蟠螭纹", "鸟纹", "波带纹"],
    "秦": ["几何纹", "菱形纹"],  # 秦代纹饰简洁
    "汉": ["四神纹", "卷云纹", "博山炉纹", "铺首衔环"],
    "唐": ["宝相花纹", "联珠纹", "鸟兽纹", "卷草纹"],
    "宋": ["折枝花卉", "莲花纹", "缠枝纹"],  # 汝窑以素面为主
    "元": ["龙凤纹", "缠枝莲", "牡丹纹"],
    "明": ["青花海水龙", "八宝纹", "岁寒三友"],
    "清": ["万寿纹", "百花纹", "锦上添花"],
}

DYNASTY_COLOR_WHITELIST = {
    "商周": ["青铜绿", "朱砂红", "黑漆"],
    "秦": ["朱砂红", "石青", "石绿", "炭黑", "铅白"],  # 兵马俑彩绘色系
    "汉": ["漆红", "漆黑", "金箔"],
    "宋": ["天青色", "月白色", "粉青色"],  # 汝窑
    "唐": ["唐三彩（黄/绿/褐）", "秘色青"],
}

def validate_prompt_accuracy(relic_info: dict, generated_prompt: str) -> dict:
    """
    校验Prompt是否符合历史准确性
    返回: {"valid": bool, "warnings": list, "suggestions": list}
    """
    dynasty = relic_info.get("dynasty", "")
    warnings = []
    suggestions = []
    
    # 检查是否混入了其他朝代的纹饰
    for other_dynasty, patterns in DYNASTY_PATTERN_WHITELIST.items():
        if other_dynasty != dynasty:
            for pattern in patterns:
                if pattern in generated_prompt:
                    warnings.append(f"警告：{pattern} 是{other_dynasty}代纹饰，与{dynasty}代不符")
    
    # 检查颜色准确性
    dynasty_key = dynasty if dynasty in DYNASTY_COLOR_WHITELIST else None
    if dynasty_key and "color" in relic_info:
        allowed = DYNASTY_COLOR_WHITELIST[dynasty_key]
        suggestions.append(f"建议使用{dynasty}代标准色系: {', '.join(allowed)}")
    
    return {
        "valid": len(warnings) == 0,
        "warnings": warnings,
        "suggestions": suggestions
    }
```

**校验流程说明：**

```
用户输入文物信息
      ↓
generate_relic_prompt() → 生成初版Prompt
      ↓
validate_prompt_accuracy() → 朝代/纹饰/颜色校验
      ↓
有警告? → [是] → 人工复核 or 自动修正 → 重新生成
      ↓
[否] → 提交Wan 2.7 生成视频
```

> 说明：该校验层不依赖Wan 2.7 自身的文物知识，而是通过外部知识库约束Prompt输入，从源头保证历史准确性。博物馆级使用时建议开启人工复核流程。
