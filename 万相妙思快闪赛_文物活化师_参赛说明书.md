# 万相妙思+ 快闪赛 · AI文物活化师

## 作品简介

**AI文物活化师** — 基于阿里云百炼平台通义万相 Wan 2.7 的文化遗产动态影像生成工具。通过图生视频（I2V）技术，将静态文物照片转化为动态影像，实现"让沉睡的文物动起来"。

## 技术方案

### 核心能力

- **模型**：通义万相 Wan 2.7（图生视频 I2V）
- **平台**：阿里云百炼 API
- **覆盖类型**：青铜器、瓷器、书画、玉雕、丝织品、简牍等6大类型

### 生成流程

1. 上传文物高清图片
2. 输入活化提示词（动作、场景、氛围）
3. 调用 Wan 2.7 I2V API 生成视频
4. 输出 5 秒动态影像

### 五大活化风格

| 风格 | 适用场景 | 提示词特点 |
|:---|:---|:---|
| 纪录片风 | 历史还原、学术展示 | 沉稳庄重，光影写实 |
| 奇幻穿越风 | 创意传播、科普 | 奇幻光效，动态特效 |
| 古代复原风 | 历史研究、教育 | 古风场景，人物还原 |
| 粒子光效风 | 展览展示、沉浸体验 | 粒子特效，光影流转 |
| 博物馆展示风 | 展品介绍、导览 | 环绕展示，细节放大 |
